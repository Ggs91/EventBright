## II - Installation

+ Clone the project: open a terminal and type in
```
git clone https://github.com/Ggs91/EventBright.git
```
+ Enter the repository:
```
cd EventBright
```
+ Download dependencies:

```
bundle install
```

+ Setup database:
```
rails db:create
rails db:migrate
rails db:seed
```
